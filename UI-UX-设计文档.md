# 1. 设计概述

## 1.1 设计目标

本设计文档旨在为企业项目全流程管理数据系统提供完整的用户界面和用户体验设计方案，确保系统具备清晰的导航结构、统一的界面风格、专业的数据展示，以及良好的用户体验。

## 1.2 设计原则

- **清晰性**: 界面布局清晰，信息层级分明，用户能够快速找到所需功能
- **一致性**: 保持整个系统的视觉风格和交互模式一致，降低用户学习成本
- **高效性**: 优化用户操作路径，提供快捷操作，提升工作效率
- **专业性**: 界面设计专业、稳重，符合企业级应用的特点
- **可访问性**: 关注不同用户群体的使用需求，确保系统易于使用

## 1.3 目标用户

| 用户群体 | 使用场景 | 设计关注点 |
|----------|----------|------------|
| 销售人员 | 客户管理、商机跟进、合同签订 | 销售漏斗、快捷录入、快速查询 |
| 项目经理 | 项目管理、任务分配、进度跟踪 | 项目概览、进度可视化、团队协作 |
| 管理人员 | 审批决策、数据分析、团队管理 | 审批效率、数据洞察、决策支持 |
| 财务人员 | 预算管理、成本核算、回款跟踪 | 财务报表、数据准确性、预算对比 |

+----------------+--------------------------------------------------+
|                |                                                  |

|   (240px)     |                                                  |
|                |                                                  |
|                |                                                  |
|                |                                                  |

+----------------+--------------------------------------------------+
```

### 2.1.1 顶部导航栏 (Header)

| 属性 | 说明 |
|------|------|
| 高度 | 64像素 |
| 背景色 | #1E293B (深蓝灰色) |
| 内容 | Logo + 系统名称 + 搜索框 + 消息通知 + 用户头像 + 下拉菜单 |

**顶部导航栏布局**:

```

|  [Logo]  企业项目管理系统    |  [搜索框]  | [消息] [帮助] [用户] |

- **图表类型**: 水平漏斗图

- **颜色**: 从左到右颜色渐变，浅到深

### 4.1.5 待办事项

- **列表形式**: 最新8条
- **内容**: 审批类型 + 业务名称 + 待办时间
- **操作**: 点击跳转至审批处理页面
- **图标**: 不同审批类型显示不同图标

### 4.1.6 近期动态

- **时间线形式**: 显示最近10条操作记录
- **内容**: 操作人 + 操作类型 + 业务对象 + 时间
- **自动刷新**: 每5分钟刷新一次

# 5. 核心交互流程设计

## 5.1 登录流程

```
+------------------------------------------------------------------+
|                                                                  |
|                    +------------------+                        |
|                    |                  |                        |
|                    |   [Logo/图标]    |                        |
|                    |                  |                        |
|                    |  企业项目管理系统  |                        |
|                    |                  |                        |
|                    +------------------+                        |
|                                                                  |
|                    用户名: [____________]                       |
|                                                                  |
|                    密码:   [____________]                       |
|                                                                  |
|                    [ ] 记住用户名                               |
|                                                                  |
|                    [        登录        ]                       |
|                                                                  |
|                    忘记密码? | 联系管理员                        |
|                                                                  |
+------------------------------------------------------------------+
```

**交互说明**:

1. 输入用户名和密码
2. 点击登录按钮或按Enter键
3. 验证成功跳转到仪表盘
4. 验证失败显示错误提示
5. 连续5次失败锁定30分钟

## 5.2 业务数据创建流程（通用）

```
1. 点击"新建"按钮
       ↓
2. 弹出创建表单弹窗
       ↓
3. 填写表单字段（实时验证）
       ↓
4. 点击"保存"按钮
       ↓
5. 后端验证
   ├─ 验证通过 → 显示成功提示 → 关闭弹窗 → 刷新列表
   └─ 验证失败 → 显示错误提示 →停留在当前页面
```

## 5.3 审批流程交互

```
1. 进入审批中心首页
       ↓
2. 点击"待我审批" tab
       ↓
3. 点击具体审批项
       ↓
4. 进入审批详情页面，查看审批内容
       ↓
5. 选择操作（通过/拒绝/退回）
       ↓
6. 填写审批意见（可选）
       ↓
7. 确认提交
       ↓
8. 显示操作结果，刷新待办列表
```

## 5.4 数据筛选与搜索

**搜索交互**:

1. 支持全局搜索和列表局部搜索
2. 输入关键词后自动搜索（防抖300ms）
3. 支持按Enter键触发搜索
4. 清空搜索条件时重置列表

**高级筛选**:

1. 点击"筛选"展开高级筛选面板
2. 多个筛选条件支持AND组合
3. 筛选条件可保存为预设
4. 清除全部筛选条件

## 5.5 表格交互

| 交互 | 行为 |
|------|------|
| 点击行 | 跳转详情页（可配置） |
| 悬停行 | 显示操作按钮 |
| 点击列头 | 排序（正序/倒序/取消） |
| 拖拽列头 | 调整列顺序 |
| 右键行 | 弹出快捷操作菜单 |
| 选择行 | 显示批量操作工具栏 |
| 分页 | 支持跳转页码和每页条数 |

## 5.6 表单验证反馈

| 状态 | 样式 |
|------|------|
| 正确 | 绿色勾选图标 |
| 错误 | 红色叉号 + 错误提示文字 |
| 警告 | 橙色警告图标 |
| 必填未填 | 红色星号提示 |
| 失焦验证 | 失去焦点时触发验证 |
| 实时验证 | 输入时实时反馈 |

# 7. 动效规范

## 7.1 过渡动画

| 属性 | 值 | 说明 |
|------|------|------|
| 标准时长 | 200ms | 常规交互 |
| 快速时长 | 150ms | 小元素过渡 |
| 慢速时长 | 300ms | 大区域/页面切换 |
| 缓动函数 | ease-in-out | 平滑过渡 |

## 7.2 交互动画

| 场景 | 动画效果 |
|------|----------|
| 弹窗出现 | 渐现 + 轻微放大 (scale 0.95→1) |

| 菜单展开 | 高度渐变 (accordion) |
| 列表项新增 | 滑入 + 淡入 |
| 消息提示 | 滑入 + 淡入，3秒后滑出 |
| 按钮点击 | 轻微缩小 (scale 0.98) |
| 卡片悬停 | 轻微上浮 + 阴影加深 |
| 数字变化 | 滚动动画 |
